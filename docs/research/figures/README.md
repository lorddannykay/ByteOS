# Figures for the research paper

The paper uses **TikZ** diagrams embedded in `paper.tex` for:

1. **Architecture** — Reference platform (Studio, Learn, Intelligence, Supabase).
2. **ALP** — Adaptive Learning Layer on top of an existing LMS.

## Optional: higher-quality figures

For submission or presentation you can replace the TikZ with exported images:

- **Architecture**: Export a diagram from the repo (e.g. from `ECOSYSTEM.md` or draw in Figma/draw.io) as PDF or PNG and include with `\includegraphics{figures/architecture}`.
- **Platform vs plugin**: Same for a “LMS + ALP layer” sketch.
- **Screenshots**: Add 1–2 app screenshots (dashboard, tutor sidebar, certificate) to the appendix or Section 4; place in `figures/` and reference in the LaTeX.

Keep diagram content consistent with the TikZ versions so the paper narrative stays aligned.
